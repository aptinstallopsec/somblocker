#Somtoday SOMBLOCKER
##Blokkeert logins voor meerdere accounts
![Logo](logo.png)
#GEBRUIKSWIJZE EN BENODIGDHEDEN:
### - Windows laptop / Chromebook met linux-subsysteem
### - Python 3.10.x

## INSTALLATIE:
### 1. git clone https://github.com/aptinstallopsec/somblocker
### 2. cd somblocker
### 3. pip install requirements.txt
### 4. Ga naar regel 28 in main.py en verander org.fill("") naar jouw organisatie
### 5. Klaar

## GEBRUIKSWIJZE:
### - Ga naar user.txt en voeg gebruikersnamen toe met de volgende syntax:
gebruiker
gebruiker1
gebruiker2
### - Run nu main.py
